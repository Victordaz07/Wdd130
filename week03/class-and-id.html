What is an id?

Un id identifica un único elemento en una página. Úsalo para marcar piezas puntuales que necesitan un estilo o anclaje específico.

What is a class?

Una class se utiliza en varios elementos a la vez para aplicar el mismo estilo y mantener consistencia visual a lo largo del sitio.
